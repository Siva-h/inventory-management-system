# Inventory Management System

## Project Overview

The Inventory Management System is a full-stack web application developed to manage product inventory efficiently. It allows an administrator to securely log in, manage product categories, and perform complete product management operations. The application is built using React.js for the frontend, Flask for the backend, PostgreSQL as the database, and JWT for authentication.

---

## Features

### Authentication
- Secure Admin Login
- JWT Authentication
- Protected Routes
- Logout Functionality

### Dashboard
- Total Categories
- Total Products
- Low Stock Products

### Category Management
- Add Category
- View Categories
- Update Category
- Delete Category

### Product Management
- Add Product
- View Products
- Update Product
- Delete Product
- Product Search
- Pagination
- Low Stock Indicator
